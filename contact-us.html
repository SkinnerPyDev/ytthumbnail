Report a Bug or Issue

If a tool is not working correctly (thumbnail not loading, logo extraction failing, banner not downloading, etc.), please email us with the following details so we can fix it fast:

  • The YouTube URL you were trying to use
  • Which tool you were using (thumbnail, logo, or banner)
  • Your device and browser (e.g., iPhone Safari, Android Chrome, Windows Firefox)
  • A brief description of what went wrong
